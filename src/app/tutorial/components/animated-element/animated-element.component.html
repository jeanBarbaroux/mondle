Dans le mode "Pays du jour", vous devez deviner le pays mystère. Pour cela, vous commencez sans informations. Vous devez entrer le nom d’un premier pays que vous souhaitez soumettre. Il est très peu probable que vous trouviez du premier coup. Vous aurez alors des informations concernant le pays que vous venez d’entrer:

  • Le Continent
  • La superficie
  • La population
  • L’indicatif téléphonique
  • La ou les langues parlées
  • Le nombre de frontières communes Par exemple: Si le pays du jour est le Luxembourg, en entrant France, le résultat sera de deux car la France et le Luxembourg ont une frontière commune avec l’Allemagne et la Belgique.

Ces informations sont accompagnées d’un code couleur et éventuellement aussi de flèches:

  • Vert: Les deux pays partagent la même caractéristique
  • Orange: Vous êtes relativement proche des caractéristiques. Pour les nombres: [-30% < pays du jour < +30%], pour les langues, au moins une langue en commun
  • Rouge: vous êtes loin de la solution
  • Flèche vers le haut: vous êtes trop bas
  • Flèche vers le bas: vous êtes trop haut

Lorsque vous trouvez le pays, il vous faudra attendre le lendemain pour pouvoir jouer à nouveau.
